# 👥 Team Availability Tracker Dashboard

## 📌 Project Overview

The Team Availability Tracker Dashboard is a web-based application that helps monitor and manage team member availability in real time. Users can view the availability status of team members and update their status using interactive toggle switches. The dashboard automatically updates statistics and visual indicators based on team availability.

## 🚀 Features

* View team members and their availability status
* Toggle availability between Available and Unavailable
* Real-time synchronization with the backend database
* Dynamic availability statistics
* Availability percentage tracking
* Team health status monitoring
* Progress bar visualization
* Last updated timestamp
* Success notifications for status updates
* User avatar initials
* Modern and responsive dashboard UI

## ⚙️ How to Run

### 1. Install Dependencies

```bash
npm install express cors
```

### 2. Start the Backend Server

```bash
node server.js
```

### 3. Open the Frontend

Open `index.html` in a browser or use Live Server in VS Code.
